# AI_PhotoSearch по описанию (Portable)

Это приложение позволяет искать изображения по текстовому описанию и/или названию файла с помощью нейросети CLIP.  
Работает полностью автономно — не требует установки Python и библиотек в систему (используется portable Python).

---

## Как запустить

1. Скачайте и распакуйте всю папку (например, AI_PhotoSearch) в любое удобное место.
2. В папке должны быть:
   - app.py — основной скрипт приложения
   - run_app.bat — файл для запуска (двойной клик)
   - папка system/ — portable Python с нужными библиотеками
   - (опционально) cache/ — кеш CLIP (создаётся автоматически)
   - (опционально) last_used_paths.pkl — история путей (создаётся автоматически)

3. Запустите run_app.bat (двойной клик).
   - Откроется браузер с приложением.
   - Если не открылся — скопируйте ссылку из консоли (обычно http://localhost:8501/) и вставьте в браузер вручную.

---

## Как пользоваться

- **Folder path** — укажите путь к папке с изображениями (можно скопировать через "Копировать как путь" в Проводнике Windows).
- **Image description** — введите описание, чтобы искать по смыслу (работает CLIP).
- **File name** — можно искать по части имени файла.
- **Scale** — количество колонок вывода.
- **Number of results** — сколько результатов выводить.
- **Show all** — показать всю базу изображений.
- **Save** — сохранить найденные изображения в новую папку.

---

## Вопросы и ответы

- **Python и библиотеки устанавливать не нужно** — всё уже встроено в папке system/.
- **История путей** — хранится в last_used_paths.pkl (рядом со скриптом).
- **Кеш базы** — находится в папке cache/.
- **Если не запускается:** убедитесь, что используете run_app.bat и что запуск .bat файлов разрешён системой.
- **Очистить историю:** удалите файл last_used_paths.pkl (создастся заново).
- **Можно переносить на другой компьютер:** скопируйте всю папку и пользуйтесь так же.
